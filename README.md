## cat
    ** Description **
    - Displays the contents of a file.
    - concatenates files.
    - Creates new files.
    
    - -n to display file with line numbers
    
    - Display content of a file.
      > cat <filename>
      > ex: cat demo.txt
    - Create new file.
      > cat > new.txt
