## cat

**Description**
 - Displays the contents of a file.
 - concatenates files.
 - Creates new files.

**Important Flags**
 - -n to display file with line numbers

**Q&A**
- Display content of a file.
  > cat demo.txt
- Merge two files to other file.
  > cat first.txt second.txt > merged.txt
- Create new file.
  > cat > new.txt
