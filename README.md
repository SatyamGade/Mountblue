## cat

**Description**
 - Displays the contents of a file.
 - concatenates files.
 - Creates new files.

**Important Flags**
 - -n to display file with line numbers

**Q&A**
 - Display content of a file.
  > cat demo.txt
 - Create new file.
  > cat > new.txt
